# Oracle Academy: Java Foundations

Este repositório contém os códigos, exercícios e projetos práticos que desenvolvi durante o curso **Java Foundations** da Oracle Academy.

## Objetivo do Curso
O curso aborda os conceitos fundamentais de programação, cobrindo desde a lógica de programação de computadores e estruturas de dados até os pilares da Programação Orientada a Objetos (POO), utilizando a linguagem Java.

## Tecnologias e Ferramentas
- Java SE (JDK 11 ou 17)
- Eclipse, NetBeans e IntelliJ IDEA.

## Conteúdo Desenvolvido
- **Lógica e Sintaxe:** Estruturas de controle de fluxo, loops e arrays.
- **Programação Orientada a Objetos:** Criação de classes, instanciação de objetos, uso de construtores, encapsulamento de atributos, herança e polimorfismo.
- **Resolução de Problemas:** Exercícios de fixação focados em algoritmos e lógica matemática aplicados ao desenvolvimento.
